## 🛠️ Run Locally

**Prerequisites:** Node.js

    ```bash
    git clone https://github.com/yahia/yahia-portfolio.git
    cd yahia-portfolio
    ```

    ```bash
    npm install
    ```
3.  **Run the application:**
    ```bash
    npm run dev
    ```
